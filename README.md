# Expense Tracker App

A modern Expense Tracker built using **HTML, CSS, and JavaScript** with multi-currency wallet support and local storage persistence.

---

##Features

- Add income and expense transactions
- Delete transactions
- Real-time balance calculation
- Income and expense summary
- Data saved in browser (LocalStorage)
- Clean modern dark UI design

---

## Technologies Used

- HTML5
- CSS3 (Modern UI)
- JavaScript (Vanilla JS)
- LocalStorage API

## How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Start adding transactions

No installation required.

---

## How It Works

### ➤ Add Transaction
- Enter description
- Enter amount
- Use negative values for expenses

### ➤ Currency System
- Select currency from dropdown
- Each currency acts as a separate wallet

### ➤ Data Storage
- All transactions are stored in LocalStorage
- Data remains after refresh
